# SQL Data Analysis Portfolio

## Project Overview
This project demonstrates SQL skills by analyzing a sales dataset using MySQL.

## Dataset
- Sales Dataset (CSV)
- Imported into MySQL Workbench

## Topics Covered
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (SUM, AVG, COUNT)
- CASE Statements
- Subqueries
- Joins
- Business Analysis Queries
