## SQL Query Files

### 01_database_setup.sql
- Creates and configures the database.
- Renames columns and adds calculated columns such as cost price, profit, profit margin, and order year.

### 02_basic_queries.sql
- Basic SQL queries using SELECT, WHERE, ORDER BY, LIMIT, and filtering records.

### 03_aggregate_queries.sql
- Aggregate functions including SUM, AVG, COUNT, MIN, MAX, GROUP BY, and HAVING for data analysis.

### 04_case_statements.sql
- Uses CASE statements to categorize and analyze data based on different conditions.

### 05_subqueries.sql
- Demonstrates nested queries (subqueries) to retrieve data based on the results of another query.

### 06_joins.sql
- Demonstrates SQL JOIN operations to combine data from multiple tables.
- *(If your project contains only one table, mention that JOIN examples are not applicable.)*

### 07_portfolio_queries.sql
- Business-oriented SQL queries, including:
  - Top 5 customers by sales
  - Sales by category
  - Monthly sales analysis
  - Profit analysis
  - Customer insights
